# Dashboard — Zé do Pix

Arquivos para publicar no Streamlit:

- app.py
- requirements.txt

## Como rodar local
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Importante
A planilha precisa estar como **"Qualquer pessoa com o link" (Leitor)** ou **Publicada na Web** para o Streamlit conseguir ler.
